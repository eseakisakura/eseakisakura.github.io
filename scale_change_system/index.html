Scale change system

スケールチェンジシステム対比表です。
ダイアトニックスケールのナチュラル(メジャー)、ハーモニック、メロディックを、
五度圏スケールから見た、相対コードリスト及び、相対するモードのリストです。


04coreworksによる『Scale change system』は、
クリエイティブ・コモンズ 表示 - 継承 4.0 国際 ライセンスの下に提供されています。
