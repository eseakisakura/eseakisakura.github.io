　//　Scale Change System　//


　■なめらかな転調は五度圏の下方のみ

<- up                                  				down ->
F# B E A D G |C| F Bb Eb Ab Db F# B E A D G |C|

C(M7) -> Eb(M7)であれば3つ下り転調と感じるため、順送りでok
cgdaebf+ -> e-b-fcgda

C ->Eであれば、8つ下がるのではなく、4つ上がりと感じ、逆戻りでng
cgdaebf+ -> ebf+c+g+d+a+

この場合、C -> Eb -> E -> Cとすれば滑らかにつながる

ただし上がり転調は初期化感(解決感)があるため、リスタートの演出となる。
fcgdaeb -> cgdaebf+では、解決感が発生する


　■コード選択と転調の共通性
これは、伴奏のコード選択においても同義である。
Dm7 -> 等価 G7 -> 解決 CM7 -> 順送 FM7

コルトレーンチェンジ化の場合は、
BM7  -> 順送 D7 -> 解決 GM7  -> 順送 Bb7 -> 解決 EbM7 ->  逆戻 Am7 -> 等価 D7


　■コードとスケールの対比関係
CM7でClyd、C7でCmixo、Cm7でCdor、Cm7b5でClocとなる。

このコードとスケールの親和性は、[基底音]と(倍音成分)によるもの
FM7は、[f]  c(g)  a  e(b)、CM7は、[c]  g(d)  e  b(f+)
Dm7は、[f]  c(g)  d  a(e)、Am7は、[c]  g(d)  a  e(b)


　■スケールアウト感
fcgdaebでのダイアトニックであれば、この上2つまで可能。
伴奏が'dfac' 'gbdf' 'cegb'で スケールがf c g d a e bであれば、
cgdaebf+でアウト感が出る。
gdaebf+c+ぐらいまでなら、耐える。
